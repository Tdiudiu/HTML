1.字符串（String）

外双内单 // 外单内双

var name = “He is called ‘diudiu’ .” ;

var name = ‘He is called “diudiu” .’ ;

{反例}

“ We are the so-called “ Vikings ” from the north. ”

打印不出来

添加转义字符来使用引号

var answer = “ He is called \ “ diudiu \ ” ” ;

length计算字符串的长度（标点符号，空格也算）

constructor 返回创建字符串属性的函数

prototype 语序您向对象添加属性和方法

字符串可以做为对象（不提倡）

var x = “diu” ;

var y = new String (“diu”);

typeof x //returns string

typeof y //returns object

                Method	                描述
            charAt()	        返回指定索引位置的字符
            charCodeAt()	    返回指定索引位置字符的 Unicode 值
            concat()	        连接两个或多个字符串，返回连接后的字符串
            fromCharCode()	    将字符转换为 Unicode 值
            indexOf()	        返回字符串中检索指定字符第一次出现的位置
            lastIndexOf()	    返回字符串中检索指定字符最后一次出现的位置
            localeCompare()	    用本地特定的顺序来比较两个字符串
            match()	            找到一个或多个正则表达式的匹配
            replace()	        替换与正则表达式匹配的子串
            search()	        检索与正则表达式相匹配的值
            slice()	            提取字符串的片断，并在新的字符串中返回被提取的部分
            split()	            把字符串分割为子字符串数组
            substr()	        从起始索引号提取字符串中指定数目的字符
            substring()	        提取字符串中两个指定的索引号之间的字符
            toLocaleLowerCase()	根据主机的语言环境把字符串转换为小写，只有几种语言（如土耳其语）具有地方特有的大小写映射
            toLocaleUpperCase()	根据主机的语言环境把字符串转换为大写，只有几种语言（如土耳其语）具有地方特有的大小写映射
            toLowerCase()	    把字符串转换为小写
            toString()	        返回字符串对象值
            toUpperCase()	    把字符串转换为大写
            trim()	            移除字符串首尾空白
            valueOf()	        返回某个字符串对象的原始值

4.数组（Array）

var cars = new Array( );

cars[0] = “Saab” ;

cars[1] = “Volvo” ;

cars[2] = “Bmw” ;


var cars = new Array ( “Saab” , “Volvo” , “Bmw” );


var cars = [ “Saab” , “Volvo” , “Bwm” ];





5.空（Null）

（1）变量的值设置为null来清空变量

（2）逻辑上null表示一个空对象的指针

（3）使用typeof检测时会返回string(objec)





6.未定义（Undefined）

（1）区分空对象指针与尚未定义的变量

（2）使用var声明变量但未初始化

（3）对未初始化的变量及未声明的变量使用typeof运算符均会返回undefined





7.对象（Object）

对象的属性以名称和值对的形式（name : value）

var person = { firstname : “John” , lastname : “Doe” , id : 5566 } ;

寻址方式：

name = person.lastname ;

name = person [ “lastname” ] ;
